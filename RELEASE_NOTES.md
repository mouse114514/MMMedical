# MMMedical v1.0.5

Medical training game mode mod for **Casualties Unknown Demo**.

## Installation
1. Install [BepInEx](https://github.com/BepInEx/BepInEx) into Casualties Unknown Demo
2. Drop `MMMedical.dll` into `BepInEx/plugins/`

## What's New
- Wish system: even rounds show a pixel-art UI to pick 1 of 7 items, guaranteed in that round's supply
- Starting INT set to 20
- Chinese item names in wish UI
- Bug fixes: state machine stuck at Generating, SpawnEquipment crash protection
